# 💼 Pranjal Raj - Software Engineer Portfolio

A modern, responsive, and interactive portfolio website showcasing my projects, technical skills, achievements, and professional journey as a Software Engineer. Designed with a premium glassmorphism interface, smooth animations, dark/light mode, and immersive 3D visuals to deliver an engaging user experience.

---

## 🌐 Live Demo

**Portfolio:** *https://heroic-stardust-148f66.netlify.app/*
